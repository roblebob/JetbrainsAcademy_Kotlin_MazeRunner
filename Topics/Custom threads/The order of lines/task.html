The order of lines

Here's a class that extends the Thread class and overrides the run method:

class MyThread(name: String) : Thread(name) {
    override fun run() {
        println("Hello from $name")
    }
}

fun main() {
    val t1 = MyThread("My-thread-1")
    t1.start()
