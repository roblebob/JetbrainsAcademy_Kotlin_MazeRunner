Big even letters

Implement a function that takes a String message as an argument. The method transforms all the letters having an even index to uppercase (index 0 is considered even) and returns the result.

It's guaranteed that input strings will not be empty.


Sample Input:
I love big letters

Sample Output:
I LoVe bIg lEtTeRs



Memory limit: 256 MB
Time limit: 8 seconds
