Application

The Resistance is organizing ships to confront the Empire. R2D2 has a list with the ships. The Resistance will attack in groups of 3 ships, but there could probably be a group with fewer ships. The droid should get a list with the average of the ammunition for each group of ships. You must code R2D2's AI to print a single list that solves this problem to organize the defense against the Empire.


Sample Input:
XWing-100 YWing-200 TWing-50 TIEFighter-300

Sample Output:
[116, 300]



Memory limit: 256 MB
Time limit: 8 seconds
