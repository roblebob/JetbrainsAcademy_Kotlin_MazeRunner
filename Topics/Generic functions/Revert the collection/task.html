Revert the collection

You have a class SomeCollection which stores values of some type. Define a generic function invert() inside this class which inverts this collection and prints it as a string (with a line break). You shouldn't change list here!

Example input

[hello, bonjour, guten tag]

Example output

[guten tag, bonjour, hello ]



Memory limit: 256 MB
Time limit: 8 seconds
