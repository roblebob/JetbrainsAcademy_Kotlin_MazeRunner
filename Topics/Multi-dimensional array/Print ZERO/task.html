Print ZERO

    val charsArray = arrayOf(
        arrayOf('A', 'B', 'C', 'D'),
        arrayOf('E', 'F'),
        arrayOf('G', 'H', 'I', 'J'),
        arrayOf('K'),
        arrayOf('L', 'M', 'N', 'O', 'P', 'Q'),
        arrayOf('R', 'S', 'T'),
        arrayOf('U'),
        arrayOf('W'),
        arrayOf('X', 'Y', 'Z')
    )
}

You need to print ZERO. Select the letters from charsArray and arrange them in the correct order, top down.
